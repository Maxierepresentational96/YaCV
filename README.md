## 🚀 What YaCV Does

YaCV helps you build a CV from a YAML file.

You edit your CV in your browser and see the result at once. The PDF view updates as you type, so you can check layout, spacing, and content without waiting.

It runs in your browser, so you do not need an account. You do not need to install a desktop app. You do not need to send your data to a server.

YaCV is a good fit if you want:

- A clean CV format
- Fast edits
- PDF output
- A layout that works with ATS systems
- Full control over your CV content

## ✍️ How to Make Your CV

YaCV uses YAML, which is a plain text format for structured data. If that sounds new, think of it as a neat way to write your CV so the app can format it for you.

A simple workflow looks like this:

1. Open the editor
2. Paste or type your CV data
3. Edit your name, job title, work history, skills, and links
4. Watch the preview update
5. Export or save the PDF when it looks right

## 🧱 Typical CV Sections

Most CVs in YaCV include:

- Full name
- Job title
- Contact details
- Short profile
- Work experience
- Education
- Skills
- Projects
- Certifications
- Languages

Keep each section short and clear. Use plain job titles and simple bullet points. That helps both people and ATS tools read it well.

## 📄 PDF Export

YaCV is built for live PDF rendering.

That means you can:

- Check page breaks
- Fix spacing
- See how text fits on the page
- Adjust content before you export

When your CV looks right, save the PDF from the browser or use the export option in the app.

## 🎯 Tips for an ATS-Friendly CV

To make your CV easier to scan:

- Use standard section names
- Avoid text boxes and heavy graphics
- Keep dates in a clear format
- Use simple bullet points
- Match your job history to the role you want
- Keep your CV easy to read

A clean layout often works better than a fancy one.
